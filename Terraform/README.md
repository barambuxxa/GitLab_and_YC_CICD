Необходимо создать сервисный аккаунт для Terraform, с нужными нам правами. Чтобы мы могли создавать инстансы в Yandex Cloud, нам нужно сформировать файл key.json из Yandex Cloud. Данный файл нужно положить в каталог нашего проекта Terraform.

Он будет добавлен в файл .gitignore, так как содержит конфиденциальную информацию.

Файл main.tf:
- Создаёт Container Registry, там будут храниться наши Docker-контейнеры.
- Создаём сервисный аккаунт и присваиваем ему нужные права. Права нужны для создания нужных нам инстансов.
- 2 Serverless Container для разработки и для продакшена
- Связывание Serverless Container с Container Registry

Файл provider.tf:
- Содержит id облака
- id каталога
- информацию откуда брать данные для авторизации (key.json)
